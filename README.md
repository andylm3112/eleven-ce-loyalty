# eleven-ce-loyalty

App de fidelización para **ELEVEN CE STUDIOS** (React + TypeScript + Vite + Supabase).

## Desarrollo

```bash
npm install
npm run dev
```

Crea `.env.local` con `VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY` y `VITE_APP_DOMAIN` (no lo subas a Git).

## Build

```bash
npm run build
```
