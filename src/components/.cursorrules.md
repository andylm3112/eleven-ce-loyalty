### PROMPT 4: ScanPage (después de que termine el anterior)
```
Implementa ScanPage.tsx (/scan/:clienteId):

1. Obtener clienteId de useParams()

2. Verificar sesión admin con useAdmin():
   const { admin, loading } = useAdmin()
   if (!admin && !loading) {
     sessionStorage.setItem('redirect_after_login', location.pathname)
     navigate('/login')
   }

3. Cargar datos del cliente:
   const { data: cliente } = await supabase.from('clientes')
     .select('*, negocios(*)')
     .eq('id', clienteId)
     .single()

4. Verificar expiración de puntos (mismo código que TarjetaCliente)

5. Buscar cupones activos del cliente (no expirados)

6. Mostrar:
   - Nombre del cliente
   - Círculos de puntos (igual que TarjetaCliente)
   - Lista de cupones activos con botón "Canjear"
   
7. Botón "Añadir Visita (+1)":
   const handleAddVisit = async () => {
     setDisabled(true)
     
     // Verificar 1 punto/día
     if (cliente.ultima_visita) {
       const hoy = new Date().toDateString()
       const ultima = new Date(cliente.ultima_visita).toDateString()
       if (hoy === ultima) {
         alert('Ya sumaste una visita hoy')
         setDisabled(false)
         return
       }
     }
     
     // Sumar punto
     await supabase.from('clientes').update({
       puntos: cliente.puntos + 1,
       ultima_visita: new Date()
     }).eq('id', clienteId)
     
     // Insertar historial
     await supabase.from('historial_visitas').insert({
       cliente_id: clienteId,
       admin_id: admin.id,
       puntos_sumados: 1
     })
     
     // Contar total visitas
     const { count } = await supabase.from('historial_visitas')
       .select('*', { count: 'exact', head: true })
       .eq('cliente_id', clienteId)
     
     // Buscar cupón para este hito (visita 5 o 10)
     const { data: cupon } = await supabase.from('cupones')
       .select('*')
       .eq('visita_requerida', count)
       .eq('activo', true)
       .single()
     
     if (cupon) {
       // Otorgar cupón
       const expiracion = new Date()
       expiracion.setDate(expiracion.getDate() + cupon.dias_validez)
       
       await supabase.from('cupones_clientes').insert({
         cliente_id: clienteId,
         cupon_id: cupon.id,
         fecha_expiracion: expiracion
       })
       
       confetti({ particleCount: 100, colors: ['#D4AF37'] })
       alert(`¡Ganaste: ${cupon.nombre}!`)
     }
     
     // Recargar cliente
     location.reload()
     
     setTimeout(() => setDisabled(false), 3000)
   }

8. Botón "Canjear Cupón" (por cada cupón):
   const handleRedeem = async (cuponClienteId, cuponId) => {
     setDisabled(true)
     
     // Marcar como canjeado
     await supabase.from('cupones_clientes').update({
       canjeado: true,
       fecha_canje: new Date()
     }).eq('id', cuponClienteId)
     
     // Insertar historial
     await supabase.from('historial_premios').insert({
       cliente_id: clienteId,
       admin_id: admin.id,
       cupon_id: cuponId,
       puntos_al_canjear: cliente.puntos
     })
     
     // Si es cupón de visita 10, resetear puntos
     const { data: cupon } = await supabase.from('cupones').select().eq('id', cuponId).single()
     if (cupon.visita_requerida === 10) {
       await supabase.from('clientes').update({ puntos: 0 }).eq('id', clienteId)
     }
     
     confetti({ particleCount: 150, colors: ['#D4AF37', '#FFD700'] })
     
     setTimeout(() => location.reload(), 2000)
   }

9. Botón "Volver" → navigate('/admin')

ESTILOS: Negro, dorado, Oswald, responsive.
Deshabilitar botones 3 segundos tras acción.
```
